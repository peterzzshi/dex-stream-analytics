# AI Context Files

This directory contains context files for AI development tools. These files are not tracked in git and are for local development assistance only.

## Files

- **project-context.md** - Technical reference and quick facts for AI tools
- **dex-stream-analytics-prompt.md** - AI assistant prompt and guidelines

## Usage

These files help AI assistants understand the project structure, technical decisions, and current implementation state. They are automatically loaded by compatible AI development tools.

## Note

Do not reference these files in project documentation or commit them to version control.
